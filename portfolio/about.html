Who I Am

I'm a self-taught developer based in Canton, Ohio. I don't come from a computer science background — I learned by building things, breaking them, and figuring out why. Every project on this site was written by hand, line by line, with a genuine understanding of how it works.

How I Work

I believe the best way to learn anything is to build something real with it. I don't follow tutorials passively — I set a goal, figure out what I need to know, and learn it in context. That approach has taken me from a blank HTML file to complete games, tools, and interactive experiences.

What I've Built

Soul Harvest A dark fantasy idle clicker with upgrades, prestige mechanics, achievements, and a lore system. Built in a single HTML file.
Soul Descent A real-time RPG dungeon crawler with combat, loot, a skill tree, and a save system. Built across 8 JavaScript files.
The Ritual A generative art piece using HTML Canvas — animated sigils, particle systems, and procedural audio.
